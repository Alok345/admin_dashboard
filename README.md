## 🚀 Getting Started

### Prerequisites
Ensure you have a modern web browser installed. No complex build tools are required for this static template.

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alok345/admin_dashboard.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd admin_dashboard
   ```

### Usage
Simply open `index.html` in your favorite web browser to view the dashboard interface:
```bash
# Using Python (Simple HTTP Server)
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your browser.
